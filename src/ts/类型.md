# TypeScript 数据类型

## 基本数据类型：
```ts
// 字符串
const string: string = "字符类型"
// 数值类型
const number: number = 10
// 布尔类型
const boolean: boolean = true
// undefined类型
const undefined: undefined = undefined
// null 类型
const null: null = null
// symbol类型
const symbol: symbol = Symbol('symbol')
```
